About Tweebot

Tweebot was created by Alexander Ilyin and is built on Django. Its purpose is to expand the existing features of Tweepy

Features:

  • Seamless integration between Tweepy and PostgreSQL. Allows the user to stream Tweets directly to a database
  • Provides an overview of streamed tweets, including summary statistics and visualizations
  • Employs a Keras Recurrent Neural Network to learn from real Twitter data. Models are trained on different categories of Tweets, and are called using a REST API structure
